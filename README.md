# The Next Vue Framework

> I'm kidding, this is just a way to learn Nitro with Vite!

## Install dependencies

```bash
pnpm install
```

## Run the app

```bash
node ./bin/cli.mjs dev
```

## Build the app

```bash
node ./bin/cli.mjs build
```
